# 1. Installation

## 1.1. Dependencies
```bash
bash

# For getip
curl

# For genpasswd
openssl

# For unextract
tar
unzip
unrar
7z
```

## 1.2. Installation
```bash
git clone https://github.com/r00tww/scripts.git
cd scripts
chmod +x install.sh
./install.sh
```
